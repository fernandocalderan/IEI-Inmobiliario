# IEI™ — Índice de Evaluación Integral

## Definición institucional
IEI™ es un framework propietario transversal para evaluar escenarios de decisión de forma determinista, auditable y accionable.

Descriptor corto:
- Framework propietario de análisis determinista, versionado y auditable.

Firma estándar:
- Powered by IEI™

## Principios
1. Determinismo
- Misma entrada, mismo resultado.

2. Integralidad
- Evalúa factores múltiples y su interacción.

3. Transparencia
- Reglas y umbrales explicitables.

4. Versionado
- Separar versión de motor y versión de framework.

5. Accionabilidad
- Entregar lectura operativa, no solo score.

## Arquitectura por verticales
- Seguridad
- Inmobiliario
- Otras verticales futuras

El framework IEI™ es común; cada vertical define sus reglas de dominio.

## Reglas de uso de marca
- IEI no es marca comercial principal frente al propietario final.
- Debe aparecer como credencial tecnológica: `Powered by IEI™`.
- En admin, documentación y reporting técnico sí puede mostrarse como `IEI™ Framework`.

## Nota legal
IEI™ es un modelo determinista basado en reglas; no es tasación oficial.
